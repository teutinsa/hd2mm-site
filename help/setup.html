So you have downloaded your desired version and want to start using it? Then this is the section for you!

First of.
If you haven't extracted the zip file you've received form downloading you should take a look at the installation guide.

Let's quickly go over what we want to achieve here.
We of course want to get to the point where you can install and use mods. But we will also talk about where your mods will be stored and how to change that. As well as where mods are staged.

Now you can run the executable.
You will see the window appear after a short time.
Now go to settings.
On the settings page you'll see three text boxes. The first one labeled "Game Directory" will be empty. This is where you put the path to your Helldivers 2 installation. (For steam this would be something like C:\Program Files (x86)\Steam\steamapps\common\Helldivers 2 ) So go ahead and select where this would be for you.

So, about the "Storage Directory" and "Temporary Directory".
  • Storage Directory
    This is where the manager will store all mods and other data.
    Feel free to change this if you want.
  • Temporary Directory
    Like the name says this directory is used for temporary storage. Mod files during staging or extracted files when adding mods for example.
    It's recommended to leave this as it's default or have it be on an SSD since it will be subject to a lot of read and write events.
