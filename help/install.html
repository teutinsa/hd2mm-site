This section covers the installation process of the Mod Manager.

If you're here however, I have to assume that you have trouble following the simple 2 steps described on the NexusMods page.

Alright let's get to it.

First.
Got to the NexusMods page again and pick the version you want. The latest is of course recommended.

Good.
Now you should have a zip file in your downloads folder. Extract this files contents anywhere you want. Ideally where you'll find it again. The location really doesn't matter.
You can put it on your desktop in a folder. It is a portable applications meaning it does not need to be installed via a traditional installer.

There you go.
You have "installed" the Mod Manager. You may now also want to look into the setup guide since you're already here.
